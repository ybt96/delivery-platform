## 一、 项目概况

**项目名称**：乡里乡亲 · 地方特色局域网络物资配送平台

**一句话定位**：服务乡村社区的本地百货展示、预约配送、百商联盟与顺风车互助出行一体化微信小程序平台。

**核心痛点**：
| 痛点 | 对应模块 |
|------|---------|
| 本地商品信息不透明，不知道哪家有货、货好不好 | 本地百货广场 |
| 出行成本高，老人孩子进城不便 | 乡村顺风车 |
| 商家获客难，缺乏联合营销手段 | 百商联盟 |
| 最后三公里配送无保障 | 配送预约服务 |

**目标用户**：消费者（本地居民）、商家、配送员、顺风车车主、平台管理员

## 四、 五大核心模块速览

### 模块一：用户体系
- 微信一键登录 + 手机号注册
- 角色体系：消费者（默认）→ 申请商家/配送员/车主
- 实名认证（顺风车与商家入驻必须）
- 收货地址管理（含村落定位）

### 模块二：本地百货广场
- 商品导向浏览（类淘宝模式），支持跨店比价
- 跨店全局购物车，一键合并结算
- 商家主页 + 评价体系 + 联盟价标记

### 模块三：配送预约服务
- **班次预约机制**：上午班（10:00 截单）、下午班（16:00 截单）
- **统一运费一口价**：按收货村落到镇中心距离阶梯定价
- **总单→子单拆分**：按商家拆分备货子单 + 集散配送任务单
- 微信支付分账结算

### 模块四：百商联盟
- 商家入驻联盟，承诺最低折扣率（≥ 9.5 折）
- 村级推客（团长）跨店组合套餐 + 微信群推广
- 优惠券体系：满减券、折扣券、代金券、平台通用券
- 智能分账：商家货款 + 团长佣金 + 骑手运费 + 平台服务费

### 模块五：乡村顺风车
- **村级隔离准入**：行程信息仅同村/社区圈内可见
- 场景化出行模板：老人就医、儿童接送、镇上赶集
- 0 佣金公益属性，支持免费互助或油费分摊（≤ ¥50/人）
- 全实名展示 + 行程分享 + 紧急报警
