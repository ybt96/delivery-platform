## 一、产品概述

### 1.1 产品背景

农村及小城镇居民存在四大痛点：

- **信息不透明**：不知道附近哪家有货、货好不好、大概多少钱
- **出行成本高**：买东西要进城，老人孩子出行不便
- **商家获客难**：本地好商家缺乏曝光渠道，依赖口耳相传
- **网购困难**：老年人不会网上购物，想买网上的东西无从下手

本平台旨在打造一个**服务本地社区的综合生活服务平台**，解决信息不对称、配送不便、出行不安全、网购困难四大核心问题。

### 1.2 产品目标

| 目标 | 衡量指标 |
|------|---------|
| 商家信息全覆盖 | 覆盖本地 80% 以上商家 |
| 配送服务覆盖 | 日均配送订单 ≥ 50 单 |
| 顺风车安全运行 | 事故率为 0，差评率 < 5% |
| 用户活跃 | 月活用户 ≥ 500 人 |

### 1.3 用户群体

| 角色 | 描述 | 核心诉求 |
|------|------|---------|
| **消费者** | 本地居民，覆盖各年龄段 | 找货方便、配送上门、价格透明 |
| **商家** | 本地各类商户 | 获客引流、管理订单、展示口碑 |
| **配送员** | 兼职或专职配送 | 接单灵活、收入清晰 |
| **顺风车车主** | 有私家车的本地村民 | 顺路搭人、分摊油费 |
| **站长** | 社区服务站负责人（村小卖部/热心村民） | 代购代办赚取佣金、服务乡邻 |
| **管理员** | 平台运营人员 | 商家审核、数据监控 |

### 4.2 核心业务规则

#### 价格展示规则

| 场景 | 展示方式 | 示例 |
|------|---------|------|
| 单一价格 | 直接显示售价 | ¥5.00/斤 |
| 价格区间 | 显示最低～最高价 | ¥3~8/斤（因时令浮动） |
| 联盟优惠价 | 标注原价+联盟折扣价 | ~~¥10~~ 联盟价 ¥8 |

#### 商品状态规则

| 状态 | 前端展示 | 是否可下单 |
|------|---------|-----------|
| 上架中 | 正常显示 | ✅ 可下单 |
| 已下架 | 显示"暂时售罄" | ❌ 不可下单 |
| 库存不足（<5） | 显示"仅剩 X 件" | ✅ 可下单 |
| 库存为 0 | 显示"已抢完" | ❌ 不可下单 |

#### 评价规则

- 收货后 **7 天内**可评价
- 超过 7 天未评价**自动好评**（5星，内容为"用户未填写评价"）
- 评价后商家可**回复一次**
- 评价内容最多支持上传 **6 张图片**

## 五、模块三：配送预约服务

### 5.1 功能清单

| 功能点 | 优先级 | 说明 |
|--------|--------|------|
| 确认订单页 | P0 | 地址选择、商品清单、优惠券、支付方式 |
| 预约配送时间 | P0 | 今天/明天/后天，选择时间段（上午/下午/晚上） |
| 立即配送 | P0 | 不预约，尽快配送（商家接单后派单） |
| 支付 | P0 | 微信支付 / 支付宝 / 货到付款 |
| 订单状态跟踪 | P0 | 实时查看订单当前状态 |
| 配送员位置跟踪 | P1 | 地图上实时显示配送员位置 |
| 订单列表 | P0 | 全部/待付款/配送中/已完成/已取消 |
| 订单详情 | P0 | 商品清单、配送信息、费用明细 |
| 催单功能 | P1 | 配送超时可催单（每笔订单限催 1 次） |
| 取消订单 | P0 | 未接单前可取消（已付款自动退款） |
| 申请退款 | P1 | 接单后取消需商家同意，24小时内处理 |
| 评价订单 | P1 | 对商品、配送员分别评价 |
| 再次购买 | P2 | 历史订单一键复购 |

```
  │     ○ 立即配送（30~60分钟）   │
  │     ○ 预约配送（选日期+时段） │
  │  4. 优惠券（可选）            │
  │  5. 买家备注（可选）          │
  │  6. 费用明细（商品+配送费）   │
  │  7. 支付方式选择              │
  └───────────────────────────────┘
        │ 点击 [提交订单]
        ▼
  ┌─ 支付页 ──────────────────────┐
  │  调起微信/支付宝支付           │
  │  → 支付成功 / 支付失败        │
```

### 5.3 预约时间规则

| 日期 | 时间段 | 可选条件 |
|------|--------|---------|
| 今天 | 上午 08:00-12:00 | 当前时间 < 10:00 |
| 今天 | 下午 12:00-18:00 | 当前时间 < 16:00 |
| 今天 | 晚上 18:00-21:00 | 当前时间 < 19:00 |
| 明天 | 上午/下午/晚上 | 全部可选 |
| 后天 | 上午/下午/晚上 | 全部可选 |

> **注**：每个时间段配送员可承接订单数有上限（默认 20 单），满员后自动置灰不可选。

### 5.4 配送费规则

| 配送距离 | 配送费 | 备注 |
|----------|--------|------|
| 0 ~ 2km | ¥2.00 | 基础费用 |
| 2 ~ 5km | ¥4.00 | — |
| 5km 以上 | 协商定价 | 配送员与用户线上确认 |
| 百商联盟订单满 ¥30 | 免配送费 | 联盟专属优惠 |

### 5.5 订单状态流转

```
待付款(0)
    │
    ├── [超时30分钟] ──→ 已取消(5)
    │
    └── [完成支付] ──→ 已支付(1)
                            │
                            ├── [主动取消] ──→ 已取消(5)，自动退款
                            │
                            └── [商家接单] ──→ 待配送(2)
                                                    │
                                                    └── [配送员取货] ──→ 配送中(3)
                                                                              │
```

## 六、模块四：百商联盟

### 6.1 模块背景与目标

百商联盟是平台的核心商业模式之一，旨在通过**商家抱团、集体让利**的方式吸引消费者留在平台内消费，形成良性循环。

- **对商家**：曝光更多、客源更稳、联盟内互相引流
- **对消费者**：在联盟商家购物享受专属折扣，产生"来这里买更划算"的黏性
- **对平台**：提升订单量，增加GMV，建立本地商业生态

### 6.2 功能清单

| 功能点 | 优先级 | 说明 |
|--------|--------|------|
| 商家入驻申请 | P0 | 填写店铺信息，上传营业执照，等待审核 |
| 联盟协议签署 | P0 | 线上确认联盟规则，承诺最低折扣率 |
| 联盟商家展示 | P0 | 首页专区展示所有联盟成员和其代表商品 |
| 联盟优惠标记 | P0 | 商品列表/详情页显示"联盟价"标签 |
| 优惠券创建 | P1 | 商家自主创建满减/折扣/代金券 |
| 平台通用券 | P1 | 平台统一发放，可在所有联盟商家使用 |

### 6.3 核心业务规则

#### 入盟规则

| 规则项 | 说明 |
|--------|------|
| 入盟条件 | 已通过平台商家认证，营业执照有效 |
| 承诺折扣 | 至少承诺联盟商品享受 **9.5 折**优惠（可更低） |
| 协议期限 | 默认签署 **6 个月**，到期可续签或退出 |
| 违约处理 | 被用户投诉"联盟价与实际不符"超过 3 次，强制退出联盟并公示 |
| 退出规则 | 协议期内不可主动退出；期满前 30 天可申请不续签 |

#### 优惠券规则

| 优惠券类型 | 规则说明 | 示例 |
|-----------|---------|------|

| **折扣券** | 全单按折扣率打折 | 享 9 折 |
| **代金券** | 无门槛直接抵扣 | 满 0 减 3（新用户礼包） |
| **平台通用券** | 所有联盟商家通用 | 平台活动发放 |
| **商家专属券** | 仅限该商家使用 | 商家自主创建 |

#### 领券限制规则

- 同一用户同一券种**每人限领 1 张**（部分活动可设每人多张）
- 优惠券有**使用截止日期**，过期自动失效
- 用券后**不支持退还**（退款订单自动退回优惠券）
- 优惠券**不可转让**，不可提现

## 七、模块五：乡村顺风车

### 7.1 模块背景与目标

乡村顺风车是平台最具特色的功能，专为解决农村出行痛点而设计：

- **接送孩子上下学**：父母忙时，邻居顺路带孩子
- **老人就医出行**：无私家车老人，约邻居顺路送医院
- **赶集出行**：同村人一起顺路进城采购

**核心原则**：
- 🔒 **仅限熟人**：同村（社区）圈内才能互相看到行程
- ✅ **实名认证**：车主和乘客均须实名，保障安全
- 🆓 **公益属性**：鼓励免费顺路，允许收取少量油费分摊
- 🛡️ **安全第一**：行程分享、紧急报警、评价体系多重保障

### 7.2 功能清单

- 陌生人（圈外人）**搜索不到任何行程**
- 乘客联系车主通过**平台内消息**，不直接显示真实电话

## 八、模块六：社区服务站（代购代办）

### 8.1 模块背景与目标

在农村和乡镇，大量中老年居民**不会使用手机网购**，但他们对网络商品有真实需求（如子女衣服、农资配件、家电、药品等）。同时，本地配送的"最后一公里"成本高、效率低。

**社区服务站**通过在每个村/街道/社区设置线下站点，解决两大核心问题：
- 🧓 **代购代办**：站长帮助不会上网的村民在网上代下单采购商品
- 📦 **自提配送**：商品集中到站点，村民自取或由站长末端配送

### 8.2 功能清单

| 功能点 | 优先级 | 说明 |
|--------|--------|------|
| 站点入驻申请 | P0 | 村民/小卖部申请成为站点，填写地址和联系方式 |
| 站点审核管理 | P0 | 管理员审核站点资质，通过后挂牌运营 |
| 站点列表展示 | P0 | 用户可查看附近的社区服务站，按距离排序 |
| 站点详情页 | P0 | 展示站点地址、营业时间、站长信息、可提供的服务 |
| 代购下单 | P0 | 站长帮村民在网上（淘宝/拼多多/京东等）代下单 |
| 代购订单记录 | P0 | 记录每笔代购订单的商品、金额、状态、村民信息 |
| 自提通知 | P0 | 商品到站后，通知村民前来自提（短信/电话） |
| 末端配送 | P1 | 站长将商品从站点配送到村民家中 |
| 代办服务 | P1 | 代缴话费、电费等生活缴费服务 |
| 站点收支统计 | P0 | 站长查看代购佣金、配送费、服务费等收入明细 |
| 站长评价 | P1 | 村民可对站长的服务进行评价 |
| 站点公告板 | P2 | 站长发布到货通知、社区公告等信息 |

### 8.3 核心业务规则

#### 代购服务流程

```
村民到站点 / 电话联系站长
    │
    ▼
描述需要购买的商品（口述 / 给图片 / 给链接）
    │
    ▼
站长在网购平台找到商品，确认价格
    │
    ▼
村民确认并付款（现金 / 微信转账给站长）
    │
    ├── 商品费用（原价）
    └── 代购服务费（按规则收取）
    │
    ▼
站长在网购平台下单，收货地址填站点地址
    │
    ▼
站长在平台录入代购订单（商品名/金额/村民/快递单号）
    │
    ▼
商品到达站点 → 站长签收 → 通知村民
    │
    ├── 村民到站自提 → 订单完成
    └── 站长末端配送上门 → 村民签收 → 订单完成（额外收配送费）
```

#### 代购服务费规则

| 商品金额区间 | 代购服务费 | 说明 |
|-------------|-----------|------|
| ¥0 ~ ¥50 | ¥3.00 | 基础服务费 |
| ¥50 ~ ¥200 | ¥5.00 | — |
| ¥200 ~ ¥500 | ¥10.00 | — |
| ¥500 以上 | 商品金额的 2% | 最高不超过 ¥50 |

#### 末端配送费规则

| 距离 | 配送费 | 说明 |
|------|--------|------|
| 站点周边 500m 内 | 免费 | 鼓励自提 |
| 500m ~ 2km | ¥2.00 | — |
| 2km 以上 | ¥5.00 | — |

#### 自提规则

- 商品到站后通知村民，**7 天内**需完成自提
- 超过 7 天未自提，站长电话提醒一次
- 超过 **14 天**仍未自提，按滞留件处理（联系村民协商退回或继续保管，收取 ¥1/天 保管费）

## 十、非功能需求

### 9.1 性能要求

| 指标 | 要求 |
|------|------|
| 商品列表首屏加载 | < 2 秒 |
| 订单提交响应时间 | < 1 秒 |
| 配送位置更新频率 | 每 10 秒更新一次 |
| 图片加载（CDN） | < 1 秒（压缩后 ≤ 200KB） |

### 9.2 安全要求

| 类别 | 要求 |
|------|------|
| 数据传输 | 全链路 HTTPS 加密 |
| 用户鉴权 | JWT Token，15 分钟过期，Refresh Token 7 天 |
| 敏感信息 | 手机号、身份证号 AES 加密存储，前端展示脱敏（138****0001） |
| 支付安全 | 走官方支付 SDK，不存储银行卡信息 |
| 接口限流 | 每 IP 每分钟最多 100 次请求 |

### 9.3 兼容性要求

| 平台 | 要求 |
|------|------|
| 微信小程序 | 支持，基础库版本 ≥ 2.10 |
| H5 浏览器 | 支持 iOS Safari 13+、Android Chrome 80+ |
| 屏幕尺寸 | 适配 320px ~ 428px 宽度 |

### 9.4 隐私保护

- 顺风车行程位置数据**仅行程参与方可见**，陌生人不可查询
- 用户真实手机号**不对外展示**，商家/配送员通过平台中转联系
- 用户可申请**注销账号**，注销后 30 天内数据完全清除

| 11 | 站长是否需要缴纳保证金？金额多少？ | 社区服务站 | 中 |
