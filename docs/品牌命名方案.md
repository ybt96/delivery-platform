## 推荐方案：优伴庭 (YBTing)

**域名**：ybting.com  
**中文名**：优伴庭  
**英文/拼音**：YBTing  

### 名称解读

| 字 | 含义 | 对应业务 |
|---|---|---|
| **优** | 优选、优质 | 本地百货广场——帮用户精选本地好货好店 |
| **伴** | 陪伴、伙伴 | 商家联盟——商家互为伙伴联合经营；邻里社交——邻居互为伙伴 |
| **庭** | 家庭、庭院、大家庭 | 社区邻里——营造本地社区大家庭的归属感 |

### 品牌 Slogan 候选

1. **「优伴庭 — 好邻好店，都在身边」**（推荐）
2. 「优伴庭 — 本地好生活，从这里开始」
3. 「优伴庭 — 你的社区生活圈」

### 品牌调性

- 🏘️ **温暖社区感**：不是冷冰冰的电商，而是邻里之间有温度的本地生活圈
- 🤝 **联盟共建感**：商家是伙伴，居民是家人，大家共同组成"优伴庭"这个大家庭
- 🛒 **本地精选感**：强调"优"字，本地好货、好店、好服务的品质筛选

### 业务重心

- 商家联盟
- 本地线上百货
- 社区邻里社交
- **线下社区服务站（品牌门店）**

### 使用场景

| 场景 | 名称 |
|------|------|
| 微信小程序 | 优伴庭 |
| 商家端小程序 | 优伴庭商家版 |
| **线下社区服务站** | **优伴庭·XX村站 / 优伴庭·XX社区站** |
| 用户口头传播 | "上优伴庭看看，附近哪家有卖的" |

| 站点宣传 | "优伴庭就在家门口，下单自取更方便" |

### 线下品牌门店规划：优伴庭·社区服务站

> 「优伴庭」中的"庭"字天然具有"院落、驿站、家园"意象，非常适合作为线下社区服务站点的品牌名。每个站点就像社区里的一个小家庭客厅，是邻里聚集、收发货物、获取服务的温暖据点。

#### 1. 站点定位

**优伴庭·社区服务站** = 线上小程序的线下触达终端

- 🏪 **不是传统门店**，而是社区服务站/自提点/便民驿站
- 📦 **核心功能**：商品自提 + 末端配送 + 社区信息枢纽
- 🤝 **运营模式**：可与村里小卖部、快递点、便民服务中心合作共建，轻资产运营

#### 2. 站点命名体系

| 层级 | 命名格式 | 示例 |
|------|---------|------|
| 镇级中心站 | 优伴庭·XX镇站 | 优伴庭·龙泉镇站 |
| 村级服务站 | 优伴庭·XX村站 | 优伴庭·白杨村站 |
| 街道/社区站 | 优伴庭·XX社区站 | 优伴庭·幸福社区站 |
| 学校/医院周边站 | 优伴庭·XX便民点 | 优伴庭·镇中心小学便民点 |

#### 3. 站点核心功能

| 功能 | 说明 |
|------|------|
| 📦 **商品自提** | 村民线上下单，到站点自提，省去逐户配送成本 |
| 🚚 **末端配送** | 站点人员负责本村/本社区范围内的上门配送（覆盖老人、行动不便人群） |
| 📋 **代下单服务** | 帮不会用手机的老人代下单、代查询商品信息 |
| 📢 **社区信息板** | 张贴本地商家优惠、顺风车信息、社区公告 |
| 🎫 **联盟优惠兑换** | 百商联盟优惠券的线下核销点 |
| 📮 **快递代收** | 可兼营快递代收发，增加站点流量和收入 |

#### 4. 站点运营模式

```
┌─────────────────────────────────────────────┐
│              优伴庭·总平台                    │
│         (ybting.com / 小程序后台)              │
└──────────────────┬──────────────────────────┘
                   │ 订单下发 / 信息同步
        ┌──────────┼──────────┐
        ▼          ▼          ▼
  ┌──────────┐ ┌──────────┐ ┌──────────┐
  │ 优伴庭   │ │ 优伴庭   │ │ 优伴庭   │
  │ 白杨村站 │ │ 幸福社区站│ │ 龙泉镇站 │
  └────┬─────┘ └────┬─────┘ └────┬─────┘
       │            │            │
   村民自提      村民自提      村民自提
   站点配送      站点配送      站点配送
   代下单        代下单        代下单
```

**合作方式（轻资产优先）**：

| 模式 | 说明 | 适用场景 |
|------|------|---------|
| 🏠 **合作共建** | 与现有小卖部/快递点合作，挂牌"优伴庭·XX站"，店主兼任站长 | 初期首选，成本最低 |
| 🏪 **自营站点** | 平台自建站点，雇佣专职站长 | 业务量大的核心镇级站点 |
| 👤 **村民代理** | 招募村里热心人/闲置劳动力担任站长，按单提成 | 偏远小村 |

#### 5. 站点人员职责

**站长（站点负责人）**：
- 管理站点日常运营，接收平台派发的订单
- 负责本村/本社区范围内的商品配送上门
- 协助不会使用手机的老人代下单
- 维护站点卫生和商品暂存
- 收集村民需求反馈给平台

**收入来源**：
- 每笔自提订单的服务费提成
- 每笔配送订单的配送费
- 代下单服务费
- 快递代收发费用（如兼营）
- 联盟优惠券核销佣金

#### 6. 站点对品牌的价值

| 维度 | 价值 |
|------|------|
| 📍 **解决最后一公里** | 商品从商家到站点由集中配送，站点到村民由站长末端配送，大幅降低成本 |
| 👴 **覆盖数字弱势群体** | 老人不会用手机也能享受服务，站长代劳 |
| 🏷️ **品牌实体化** | 线下挂牌让"优伴庭"品牌在村镇有实体存在感，增强信任 |
| 📊 **数据采集** | 站点是了解各村真实需求的最佳渠道 |
| 🔄 **流量入口** | 站点本身成为线下获客入口，带动小程序注册和使用 |

## 备选方案

### 方案二：易帮亭 (YBTing)

- **释义**："易"代表容易、便捷，"帮"代表互助帮忙（代买代送、顺风车），"亭"古代指驿站、休息点
- **寓意**：让乡亲们互相帮忙变得更容易的本地生活服务驿站
- **适合场景**：如果后续业务重心偏向跑腿、配送、顺风车驿站等实体落地感更强的业务

### 方案三：易办挺 (YBTing)

- **释义**："易办"意为办事容易、买东西容易、出行容易，"挺"代表好、棒（方言口语化）
- **寓意**：在这个平台上干啥都容易，体验"挺"好
- **特点**：带有下沉市场口语化特征，好记、亲切

### 方案四：优便停 (YBTing)

- **释义**："优便"即优质便捷，"停"指代顺风车停靠点、物资配送站
- **寓意**：为出行和物资流转提供优质便捷的停靠服务

### 方案五：友帮听 (YBTing)

- **释义**："友帮"指亲友邻里互助，"听"指倾听老乡们的需求
- **寓意**：倾听乡镇需求，邻里互帮互助的平台
