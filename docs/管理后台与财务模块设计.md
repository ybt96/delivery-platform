# 地方特色局域网络物资配送平台 - 管理后台与财务管控模块设计

## 1. 模块定位与分层架构
针对下沉市场用户（乡镇商家、村长等）不习惯使用电脑办公的现状，系统管理权限采用**“移动端操作为主，Web端财务管控为辅”**的极简架构。

### 1.1 移动端管理 (微信小程序内嵌)
将高频的业务操作直接集成在微信小程序中，根据不同角色赋予不同权限：
*   **商家端：** 手机拍照上传商品、修改库存价格、扫码核销/接单、查看本店每日收益。
*   **村长/圈子管理员端：** 接收本村“顺风车”用户的加入申请，手机上一键审核通过/驳回；查看本村人员活跃情况。
*   **团长/推客端：** 手机端挑选联盟商品“组局”生成套餐，一键分享到微信群，实时查看个人的预估佣金。

### 1.2 Web端总后台 (仅供平台方/您和团队使用)
Web后台作为整个平台的“神经中枢”，主要聚焦于**宏观数据管控**和**复杂财务对账**。

## 2. Web端核心功能：财务分账与结算监控 (核心维度)

由于平台涉及多方利益（平台、商家、骑手、团长），财务资金的安全流转与清晰对账是重中之重。系统深度集成**微信支付的分账能力**。

### 2.1 全局财务驾驶舱 (数据大屏)
*   **实时流水监控：** 监控每日总交易额（GMV）、实收金额、退款金额。
*   **分账流向总览：** 图表化展示资金的拆分比例和走向：
    *   商家货款总额（进入各商家微信商户号/零钱）。
    *   骑手运费总额（每日统配运费支出）。
    *   团长佣金总额（百商联盟推广支出）。
    *   平台抽成/服务费（平台净收入）。

### 2.2 自动分账与结算单管理
*   **动态分账规则引擎：** 后台可灵活配置不同类目、不同联盟套餐的抽佣比例和分账规则。
*   **结算对账单：** 系统自动按天/周生成包含所有子订单的详细对账单。任何一笔“总单支付”，都能向下追溯到拆分给张三多少钱、李四多少钱。
*   **异常拦截与人工介入：** 
    *   如果发生退款，系统自动按原比例撤回各方分账资金。
    *   提供“资金冻结与解冻”功能，应对买卖纠纷。

### 2.3 基础档案与合规管控 (辅助功能)
*   **用户与商家库：** 全局查看所有实名认证用户资料、入驻商家的营业执照资质。
*   **内容巡查：** 对平台内的商品评价、顺风车备注等文本信息进行统一的违规词监控。
